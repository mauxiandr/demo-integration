# demo-integration

## Install
- npm install

## Usage
- Create an .env file with variables:
    * ISSUE_ID
    * SENTRY_TOKEN 
    * ORGANIZATION_SLUG 
    * PROJECT_SLUG 
    * STAGE_TAG
    * STAGE
    * REGION
    * FACTURADOR_TOKEN
    * S3_BUCKET 
    * ACCOUNT
- Endpoints
    - POST - /
    - GET - /download
